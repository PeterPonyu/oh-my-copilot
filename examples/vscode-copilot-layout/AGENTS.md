# VS Code Copilot Layout Instructions

This directory is a stronger VS Code Copilot workspace smoke test, not the root
workspace and not the reusable plugin package.

- Keep examples explicitly bounded to this workspace.
- Use `.github/agents`, `.github/prompts`, `.github/skills`, and `.github/hooks`
  as VS Code-oriented examples.
- Preserve the distinction between root registration, plugin assets, and this
  illustrative workspace.
- Avoid full OMC/OMX parity language unless current Copilot evidence proves the
  specific behavior.
